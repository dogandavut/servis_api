# servis_api

Bu proje, Python ve Flask kullanılarak geliştirilmiş bir teknik servis yönetim API'sidir.

## Ana Modüller
- Müşteri Yönetimi
- Paket Tanımlama ve Takip
- Servis Kayıtları
- Raporlama

## Kullanılan Teknolojiler
- Python 3
- Flask
- MSSQL (pyodbc ile)
- VS Code
